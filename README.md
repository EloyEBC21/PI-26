# Proyecto Intermodulas 2025-2026 by Eloy Ballester Chust

Este proyecto consiste en un gestor de cartas del mundo del TGC (Trading Game Card).
Que consiste en una página donde muestro como comprar/vender/revisar/inspeccionar tanto
cartas como sus mismos precios a de mas del idioma de estas.

El planteamiento de este proyecto fue que siempre que reviso paginas web solamente encuentro páginas
que hacen solamente 1 función de estas por lo que yo pensé en juntarlas todas y probar que
se puede juntar todo y hacer un buen proyecto.
